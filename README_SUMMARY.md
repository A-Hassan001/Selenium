# Selenium Automation Projects - Summary

## Repository Overview

This collection contains three comprehensive Selenium-based automation projects:

### 1. **EXPEDIA HOTEL** 
Automated hotel booking and search functionality on the Expedia platform. Handles hotel searches, filtering, result navigation, and web scraping of hotel data.

### 2. **HKJC Racing Time Scraper**
Web scraper for Hong Kong Jockey Club racing data. Automates extraction of racing schedules, race times, and timing information from the HKJC website.

### 3. **Trademarks IPO Gov UK Automation**
UK Government IPO trademark database automation. Automates trademark searches, batch processing of companies, and generates reports on trademark registration status.

## Key Technologies

- **Python** - Primary programming language
- **Selenium WebDriver** - Web automation framework
- **Undetected Chrome** - Anti-detection browser automation
- **Chrome/Firefox/Edge** - Browser support

## Common Features

- Automated web interactions
- Data extraction and scraping
- Batch processing capabilities
- Error handling and retry logic
- Comprehensive logging
- Cross-platform support

## Prerequisites

- Python 3.7 or higher
- Chrome or Firefox browser
- WebDriver (ChromeDriver or GeckoDriver)
- pip (Python package manager)
